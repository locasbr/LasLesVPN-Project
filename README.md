# 🚀 LaslesVPN - Landing Page

![LaslesVPN Logo](src/images/logo.png)

Uma landing page moderna e responsiva para o serviço de VPN LaslesVPN, desenvolvida com HTML, CSS e JavaScript. Este projeto demonstra habilidades em design web, animações e estruturação de conteúdo para uma página de vendas eficaz.

## ✨ Características

- **Design Responsivo**: Otimizado para dispositivos móveis, tablets e desktops
- **Animações Suaves**: Efeitos visuais atrativos com CSS e JavaScript
- **Carrossel Interativo**: Depoimentos de clientes com navegação intuitiva
- **Seções Organizadas**: Header, recursos, planos, mapa global e depoimentos
- **Integração com Bulma**: Framework CSS para componentes elegantes
- **Fonte Personalizada**: Rubik para uma tipografia moderna

## 🚀 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/seu-usuario/laslesvpn-project.git
   cd laslesvpn-project
   ```

2. **Instale as dependências**:
   ```bash
   npm install
   ```

3. **Abra o projeto**:
   - Abra `src/index.html` no seu navegador
   - Ou use um servidor local para melhor experiência

## 🎨 Personalização

Este projeto é altamente personalizável:

- **Cores**: Modifique as variáveis CSS em `estilos/style.css`
- **Conteúdo**: Edite textos em `src/index.html`
- **Imagens**: Substitua as imagens na pasta `src/images/`
- **Estilos**: Ajuste os arquivos CSS individuais para cada seção
